# Rubric: exec_summary

**Domain:** summarization
**Total Points:** 38
**Pass Threshold:** 0.85

## Criterion 1: sum_compression
**Category:** structure
**Max Points:** N/A
**Description:** Achieves 20:1+ compression — exactly 3 bullets, each 1-2 sentences
**Pass Condition:** Exactly 3 bullets. Each bullet is exactly 1 sentence (no semicolons, em-dashes, or compound sentences). Total under 75 words. No hedging words ('may', 'could', 'potentially', 'appears to').

## Criterion 2: sum_fidelity
**Category:** accuracy
**Max Points:** N/A
**Description:** First bullet accurately captures the main thesis or primary finding from the source post
**Pass Condition:** Bullet 1 states the core thesis/finding in declarative form. The claim can be directly traced to a specific section of the source text.

## Criterion 3: sum_exec_value
**Category:** utility
**Max Points:** N/A
**Description:** Summary provides actionable insights with concrete implications for decision-making
**Pass Condition:** At least one bullet includes a quantified metric, timeline, or measurable outcome. At least one bullet states a clear implication using action-oriented language (e.g., 'should implement', 'reduces costs by', 'enables teams to').

## Criterion 4: sum_standalone
**Category:** clarity
**Max Points:** N/A
**Description:** Summary is self-contained — no context needed to understand it
**Pass Condition:** Doesn't reference 'the post' or 'the author'. Defines any jargon. A reader with no context gets the point.
