**Scoring Method:** `weighted_components`
**Max Points:** 10

### Sub-Attributes

| Sub-ID | Weight | Description | Measurement |

| `schema_validity` | 30% | Valid JSON Schema syntax | 1.0 if valid draft-07+, 0.5 if mostly valid, 0.0 if invalid |
